About this data

We've been using Chrome's anonymous usage statistics to count the occurrences of certain CSS features in the wild. The numbers on this page indicate the percentages of Chrome page loads (across all channels and platforms) that use the corresponding CSS property at least once. Newly added use counters that are not on Chrome stable yet only have data from the Chrome channels they're on. Data is ~24 hrs old.
