About this data

We've been using Chrome's anonymous opt-in usage statistics to count the CSS properties which are animated. A property that is animated multiple times during a single page view is counted each time it animates. Percentages on this page indicate the amount of times (as the fraction of all animated properties) each property is used in a Transition or Animation.. Data is ~24 hrs old and reflects usage across all channels and platforms.
